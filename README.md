# BC Finance Speedrun - Training Planner

## Why this exists

When an organisation adopts Microsoft Dynamics 365 Business Central, the finance and consulting teams face a steep learning curve. The platform is deep - posting groups, dimensions, sub-ledgers, GST/BAS compliance, project accounting, and Dataverse integration all need to be understood before anyone can contribute meaningfully to an implementation.

Traditional training paths (week-long courses, lengthy certification tracks) don't match the reality of digital transformation projects where teams need to be productive in days, not months.

**This resource was created to solve that gap.** It compresses the critical Business Central finance knowledge into a structured 5-day speedrun (~2 hours per day) so that:

- **New team members** can ramp up fast enough to join active BC implementations without slowing the project down.
- **Consultants crossing over** from Dynamics 365 CE or other platforms can build BC finance fluency quickly and speak the language in client workshops from week one.
- **Project teams adopting BC** have a single, self-contained checklist that ensures nobody skips foundational concepts that cause costly mistakes later (misconfigured posting groups, missing dimensions, broken bank reconciliation flows).

## What it covers

| Day | Focus |
|-----|-------|
| 1 | Foundations: Chart of Accounts, Posting Groups, Dimensions, transaction flow |
| 2 | Sub-Ledgers: AR, AP, Bank, Cash, payment application |
| 3 | GST/BAS compliance, period-end close, Financial Reports |
| 4 | Dimensions deep-dive, Budgets, Projects (Jobs) and WIP |
| 5 | NFP considerations, BC-Dataverse/CE integration patterns |

## How to use

Open `BCquicktraining.html` in any browser. The page respects your device's light/dark mode preference automatically. Check off items as you complete them - progress is saved locally.
